# 《余晖号》Afterglow Express — Game Design Document

> **版本**: v1.0  
> **日期**: 2026-05-16  
> **类型**: Roguelite × 搜打撤 (Search-Fight-Extract)  
> **目标**: 高校+企业联合 Game Jam 提交 Demo  
> **周期**: 1-2 周

---

## 1. 项目概要

### 1.1 核心概念

《余晖号》是一款 **2D 扁平斜俯视角 Roguelite 搜打撤** 游戏。玩家扮演一名「残响猎人」，驾驶磁轨母车《余晖号》，深入被电磁灾难摧毁的静默区，搜刮旧文明的残响晶片，对抗失控机械体，并在信号弹的指引下撤离。

### 1.2 核心差异化创新

- **双 Tag 物品系统（Weight + Erosion）**：每件物品有负重和侵蚀两个维度的代价
- **玩家驱动式节奏**：侵蚀、噪音、风暴三个系统相互拉扯，没有固定倒计时
- **信号弹撤离**：撤离点由玩家决定而非预设
- **GTFO 式噪音机制**：战斗产生噪音，唤醒休眠敌人
- **残响叙事 Meta**：晶片承载旧世界记忆碎片，解锁即解锁故事+微量增益

### 1.3 技术栈

| 层 | 选型 |
|----|------|
| 引擎 | Unity (C#) |
| 视角 | 2D 扁平斜俯视 |
| 移动 | 360° 自由坐标，非网格 |
| 美术 | AI 辅助像素风 (ComfyUI 本地管线) |
| 地图 | Perlin 噪声随机生成 |

## 2. 世界观

### 2.1 背景

2040 年代，一场被称为「静默」(The Silence) 的电磁灾难摧毁了全球通信网络和大部分电子设备。文明没有毁灭，而是退回到了「孤岛化」状态。旧时代的科技残骸散落在荒野中，部分 AI 和机械体脱离人类控制，形成了自己的「领地」。

主角是残响猎人——驾驶拼装磁轨车《余晖号》，穿梭于废墟之间，搜寻还能用的科技核心「残响晶片」。这些晶片是旧文明的数据碎片，也是重建世界的钥匙。

### 2.2 氛围调性

近未来轻微末世感，硬科幻外壳+温情内核。视觉主色调为 **翡翠绿 + 黄昏橙**，文明退潮但人类仍在。

## 3. 核心循环

### 3.1 一次 Run 的流程

```
母车准备 → 投放至静默区 → [搜刮/战斗/决策] → 信号弹呼叫撤离 → 等待母车 → 撤离/死亡
```

### 3.2 三阶段

| 阶段 | 行为 | 压力来源 |
|------|------|----------|
| **搜** | 探索地图、拾取物品、管理 Weight/Erosion | 迷雾视野、侵蚀累积 |
| **打** | 与失控机械体交战 | 噪音唤醒更多敌人 |
| **撤** | 发射信号弹、坚守等待母车 | 信号吸引敌人、风暴逼近 |

### 3.3 节奏：玩家驱动式

不预设难度曲线。节奏完全由玩家行为决定：
- 搜得越多 → 侵蚀越高 → 风暴追踪越激进 + 视野越窄
- 打得越多 → 噪音越高 → 唤醒更多/更强敌人
- 撤得越晚 → 压力叠加 → 翻车风险指数上升

## 4. 角色系统

### 4.1 主角

单一主角，无角色选择。通过装备更换影响 HP 和移动速度。

### 4.2 技能

| 技能位 | 来源 | 说明 |
|--------|------|------|
| 主动技能 | 角色自带 | 进图即用，CD 限制 |
| 被动技能 | 角色自带 | 持续生效 |
| 武器普攻 | 武器 | 固定射击模式 |
| 武器主动 | 武器 | 每把武器的特殊技能（如 EMP 波、蓄力射） |

---

## 5. 物品与双 Tag 系统

### 5.1 双 Tag

每个可拾取物品具有两个 Tag：

| Tag | 性质 | 规则 |
|-----|------|------|
| **Weight（负重）** | 物理限制 | 身上的物品 Weight 总和 ≤ maxWeight |
| **Erosion（侵蚀）** | 精神/生理污染 | 首次拾取时产生侵蚀值，物品侵蚀属性被移除 |

### 5.2 侵蚀规则

- **硬门槛**：`当前侵蚀 + 物品侵蚀值 > maxErosion` → 无法拾取
- **软压力**：侵蚀越高 → 静默风暴追踪强度↑、视野范围↓
- **不是死亡条件**：侵蚀满 100% 不致死，只是无法再拾取物品
- **纯局内**：不跨 Run，不滚雪球到 Meta

### 5.3 搜刮物品

| 类型 | 说明 |
|------|------|
| 残响晶片 | 核心货币，用于 Meta 解锁和局外购买 |
| 弹药 | 武器消耗品 |
| 能量电池 | 回复 HP |
| 局内武器 | 随机品质（白/绿/蓝/紫） |

---

## 6. 武器系统

### 6.1 类型

固定武器类型（如手枪、步枪、霰弹、能量武器），每种有固定射击手感和主动技能。

### 6.2 品质

局内拾取的武器随机品质：白 → 绿 → 蓝 → 紫，影响伤害、射速、弹匣、噪音产生。

### 6.3 局外获取

玩家带入 白 初始武器进入地图，通过搜刮获得更好的武器。没有 upgrade 或蓝图系统。

---

## 7. 战斗系统

### 7.1 噪音机制

- 每个敌人独立噪音条
- 开枪、使用武器主动技能、破解 loot 积累噪音，噪音积累与距离有关
- 超过阈值 -> 敌人觉醒，攻击玩家
- 玩家攻击非觉醒敌人会立即觉醒
- 噪音条缓慢消退

### 7.2 生命系统

- 经典 HP 条
- 能量电池回复
- 侵蚀不影响 HP 上限

---

## 8. 敌人系统

### 8.1 阵营

纯敌对失控机械体，无同阵营竞争 AI。

### 8.2 三型制

| 类型 | 特点 | 行为 |
|------|------|------|
| **轻型** | 血少移速快 | 随机游走在地图上/巡逻后追击 |
| **中型** | 标准属性 | 区域巡逻、主动迎敌 |
| **重型** | 血厚伤害高 | 噪音阈值触发唤醒 |

### 8.3 分布

随机生成但可形成聚落（高价值区密度更高）。

觉醒后会生成新机械体。

---

## 9. 地图系统

### 9.1 生成方式

Perlin/Simplex 噪声生成地形底图 → 算法划分高/低价值区 → 填充敌人/loot/障碍物。

### 9.2 结构

- 整张图：起始投放点 → 探索区 → 任意位置撤离
- 高价值区 = 风险高（更多/更强的机械体）
- 低价值区 = 过渡区域
- 出发点随机，撤离由玩家决定

### 9.3 战争迷雾

- 圆形视野锥（视角跟随角色）
- 走过区域留痕（永久亮）
- 未探索区域完全不可见
- 侵蚀越高 → 视野半径越小

---

## 10. 撤离系统

### 10.1 信号弹撤离

- 地图上无固定撤离点
- 玩家在任何位置发射信号弹
- 母车《余晖号》飞抵信号位置（60-90 秒等待）
- 等待期间信号吸引附近机械体涌来，还可能生成新机械体
- 母车到达后交互登车 = 撤离成功

---

## 11. 双重压力系统

### 11.1 残响负荷（Erosion）

- 拾取物品产生侵蚀
- 侵蚀越高 → 静默风暴追踪强度↑、视野范围↓
- 硬门槛阻止过度搜刮

### 11.2 静默风暴

- 雷达可视的移动预警区域
- 在地图上持续移动，会停止、变速、可能转向追踪玩家
- 被风暴吞没 → 持续伤害 + 敌人全部激活
- 风暴大小、移动速度、停止时间、路线追逐概率需 Balance

---

## 12. 难度系统

### 12.1 三级难度

| 级别 | 世界观命名 | 说明 |
|------|-----------|------|
| Lv1 | 微弱回响 | 基础体验，风暴慢、敌人少、侵蚀低 |
| Lv2 | 清晰信号 | 标准挑战，风暴适中、敌人种类增多 |
| Lv3 | 尖锐悲鸣 | 极限考验，风暴激进、重型机械体初始唤醒、敌人密度最高 |

---

## 13. Meta 成长系统

### 13.1 核心机制

残响晶片 → 解锁旧世界「记忆片段」→ 每段记忆附带条件性增益 + 微量永久数值提升。

### 13.2 增益性质

**条件性增益（主）**——改变特定情境下的表现：
- 例：「在静默风暴范围内 +10% 移动速度」
- 例：「满负荷时伤害 +15%」
- 例：「对觉醒状态的机械体 +20% 暴击」

**微量数值（辅）**——少量永久面板提升。

### 13.3 Meta 控制的属性

- `maxWeight`：仅通过 Meta 提升
- `maxErosion`：仅通过 Meta 提升
- 其他基础属性由装备决定

## 14. Run 结算

### 14.1 撤离成功

| 状态 | 结果 |
|------|------|
| 成功撤离 | 晶片全保留、装备保留、记忆碎片解锁 |

### 14.2 死亡

| 状态 | 结果 |
|------|------|
| 死亡 | 晶片全丢、带进图的装备丢失、记忆保留 |

## 16. 待 Balance 参数

| 参数 | 说明 |
|------|------|
| Weight/Erosion 基础值 | 每种物品的默认双 Tag |
| maxWeight/maxErosion 初始值 | 角色初始上限 |
| 侵蚀→风暴追踪强度映射 | 侵蚀值对应的风暴行为 |
| 侵蚀→视野半径映射 | 侵蚀值对应的迷雾范围 |
| 噪音阈值 | 唤醒敌人的噪音临界值 |
| 噪音消退速率 | 噪音自然衰减速度 |
| 信号弹等待时间 | 母车抵达秒数 |
| 风暴参数 | 大小、速度、停止时间、追踪概率 |
| 各品质武器参数 | 伤害、射速、弹匣、噪音 |
| 难度分级参数 | 每级难度的敌人密度、风暴行为、侵蚀倍率 |

## 17. 不做 (Out of Scope)

- 多角色选择
- 技能树/天赋系统
- 同阵营竞争 AI
- 武器模组/零件拼接
- 多人联机
- 剧情分支/多结局
- BOSS 战（可留作大赛后扩展）
