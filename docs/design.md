> **版本**: v3.1

## 1. 项目概要

### 1.1 核心概念

《余晖号》是一款 **3D 正交斜俯视角 Roguelite 搜打撤** 游戏。玩家扮演一名「残响猎人」，驾驶磁轨母车《余晖号》，深入被电磁灾难摧毁的静默区，破解旧文明容器搜刮残响遗物和收集品，对抗失控机械体，并在信号弹的指引下撤离。

### 1.2 核心差异化创新

- **负重 + 全局侵蚀双维压力**：物品受负重限制，侵蚀随时间和受击持续增长，双重压力交织
- **破解 Loot 系统**：容器需长按交互破解，侵蚀越高破解越慢
- **玩家驱动式节奏**：侵蚀与噪音相互拉扯，时间推移敌人不断增强
- **独立警戒值**：每个敌人独立追踪噪音，渐进式积累觉醒
- **信号弹撤离**：撤离点由玩家决定而非预设
- **事件系统**：固定地图 + 随机事件，每局体验不同

### 1.3 技术栈

| 层 | 选型 |
|----|------|
| 引擎 | Godot 4.x (GDScript) |
| 视角 | 3D 正交斜俯视 |
| 移动 | 360° 自由坐标，非网格 |
| 美术 | AI 辅助像素风 (ComfyUI 本地管线) |
| 地图 | 手工制作固定地图 + 事件系统 |

### 1.4 MVP 定义（最小可玩版本）

> Demo 核心目标：**让玩家在 5 分钟内体验到"搜-打-撤"循环的完整张力。**

MVP 必须包含的体验闭环：

```
玩家出生 → 探索地图 → 破解容器获得物品（负重积累，侵蚀随时间/受击增长）→ 遭遇/触发敌人 → 战斗消耗弹药 → 权衡"继续搜还是撤" → 发射信号弹 → 坚守等待 → 撤离成功/死亡 → 结算
```

MVP 功能边界：

| 包含（Demo 必做） | 不包含（见 §17 升级方向） |
|------|---------------------------|
| 360° 移动 + 冲刺 | 多角色 / 技能树 |
| 单武器 + 标准弹药射击 | 弹种切换 / 武器品质分级 |
| 破解容器 + 负重/侵蚀压力 | 容器重封 / Meta 成长 |
| 2 型敌人（巡逻型 + 休眠型） | 中型敌人 |
| 独立警戒值（渐进式积累觉醒） | 经济系统 / 商店 / 交易 |
| 时间刷怪（递增强度） | 局外装备选择 / 防御装备 / 投掷道具 |
| 信号弹撤离 + 坚守等待 | 多难度选择 |
| 固定地图（20-30 屏）+ 事件系统 | 母车主界面 / 展厅 / 商店 UI |
| 侵蚀影响视野 + 破解速度 | |
| 简单 HUD + 开始/结算界面 | |

## 2. 世界观

### 2.1 背景

2040 年代，一场被称为「静默」(The Silence) 的电磁灾难摧毁了全球通信网络和大部分电子设备。文明没有毁灭，而是退回到了「孤岛化」状态。旧时代的科技残骸散落在荒野中，部分 AI 和机械体脱离人类控制，形成了自己的「领地」。

主角是残响猎人——驾驶拼装磁轨车《余晖号》，穿梭于废墟之间，搜寻旧文明的残响遗物。这些物品承载着旧世界的记忆碎片，也是重建世界的钥匙。

### 2.2 氛围调性

参考《星之终途》(Stella of the End)：近未来轻微末世感，硬科幻外壳+温情内核。视觉主色调为 **翡翠绿 + 黄昏橙**，文明退潮但人类仍在。

## 3. 核心循环

### 3.1 一次 Run 的流程

```
投放至静默区 → [破解容器/搜刮/战斗/决策] → 信号弹呼叫撤离 → 坚守等待母车 → 撤离/死亡 → 结算
```

### 3.2 三阶段

| 阶段 | 行为 | 压力来源 |
|------|------|----------|
| **搜** | 探索地图、破解容器、管理负重与侵蚀压力 | 迷雾视野、侵蚀累积、破解耗时 |
| **打** | 与失控机械体交战 | 噪音积累警戒值、时间推移敌人增多 |
| **撤** | 发射信号弹、坚守等待母车 | 信号吸引敌人、时间刷怪高峰 |

### 3.3 节奏：玩家驱动 + 时间压力

节奏由玩家行为和时间双重驱动：
- 待得越久 → 侵蚀自然增长 → 视野越窄 + 破解越慢 + 敌人越强 + 刷怪越猛
- 打得越多 → 噪音越高 → 警戒值积累 → 唤醒更多敌人
- 受击越多 → 侵蚀跳升 → 加速进入高侵蚀状态
- 撤得越晚 → 压力叠加 → 翻车风险指数上升

## 5. 物品与负重系统

### 5.1 负重系统

每个可拾取物品具有 **Weight（负重）** 属性：

- 身上的物品 Weight 总和 ≤ maxWeight
- 超过负重上限 → 无法再拾取物品

侵蚀（Erosion）不是物品属性，而是**全局生存压力**（详见 §11.1）：
- 随时间自然增长 + 受击跳升
- 侵蚀达到 100% → 无法拾取任何物品

### 5.2 破解 Loot 系统

物品不是直接拾取的——大部分物品存放在**密封容器**中，需要破解才能获取内容。

- **破解方式**：站在容器前，按住交互键 2-3 秒读条。读条期间不能移动/攻击
- **侵蚀影响**：侵蚀越高，读条越慢（侵蚀 0% = 2s，侵蚀 80% = 5s），倍率线性映射
- **破解本身不产生侵蚀**
- **受伤中断**：读条期间被攻击则中断

### 5.3 容器类型与 Loot 分布

| 容器类型 | 出现概率 | 外观 | 内容 |
|----------|----------|------|------|
| **普通容器** | ~80% | 生锈储物箱、旧数据终端 | 收集品（分数来源）+ 偶尔标准弹药 |
| **弹药箱** | ~12% | 绿色军用箱，可辨识 | 标准弹药补给 |
| **医疗容器** | ~8% | 白色医疗柜，红十字标记 | 净化剂 ×1（降低侵蚀）、能量电池 |

### 5.4 侵蚀规则

侵蚀是全局生存压力机制，随玩家在局内存活时间自然增长。

**侵蚀增长：**

| 来源 | 说明 |
|------|------|
| **时间流逝** | 每 X 秒侵蚀 +1%，全程持续增长 |
| **受击** | 被敌人攻击 → 侵蚀小幅跳升（+2~3%） |

**侵蚀减少：**

| 来源 | 说明 |
|------|------|
| **净化剂** | 使用后降低侵蚀 15~20%。局内医疗容器搜刮获得 |

**侵蚀影响：**

| 侵蚀值 | 视野 | 破解速度 | 敌人 HP/伤害 | 刷怪间隔 | 刷怪池 |
|--------|------|----------|-------------|----------|--------|
| 0-25% | 正常 | 正常 | ×1.0 | 正常 | 巡逻型 |
| 25-50% | -15% | 轻微减速 | ×1.0 | ×0.85 | 巡逻型为主 |
| 50-75% | -30% | 明显减速 | ×1.1 | ×0.7 | 巡逻型 + 少量休眠型 |
| 75-99% | -50% | 严重减速 | ×1.2 | ×0.5 | 混合敌人增多 |
| 100% | 极小 | 极慢 | ×1.35 | ×0.5 | 无法拾取任何物品 |

**硬规则：**
- **硬门槛**：100% 侵蚀 = 无法拾取任何物品
- **不是死亡条件**：侵蚀满 100% 不致死
- **纯局内**：不跨 Run，每次 Run 从 0% 开始

### 5.5 物品列表

| 类型 | 说明 | Weight |
|------|------|--------|
| 收集品 | 撤出后计入结算分数，是主要得分来源 | 有 |
| 标准弹药 | 武器消耗品（有限） | 有 |
| 能量电池 | 回复 HP | 有 |
| 净化剂 | 降低侵蚀 15~20% | 有 |

## 7. 战斗系统

### 7.1 噪音与警戒值机制

**每个敌人拥有独立的警戒值（噪音条）**，不共享。

噪音产生：

| 行动 | 噪音值 | 传播范围 |
|------|--------|----------|
| 走路 | 0 | — |
| 破解容器 | 小 | 近 |
| 冲刺 | 中 | 中 |
| 开枪 | 大 | 远 |
| 信号弹发射 | 极大 | 全图 |

噪音传播：距离衰减。敌人接收到的噪音值 = 玩家噪音值 × 距离衰减系数 → 累加入该敌人的警戒值。

警戒值超阈值 → 敌人觉醒进入战斗。

攻击未觉醒敌人 → 该敌人立刻觉醒（警戒值直接拉满）。

警戒值随时间缓慢消退。

### 7.2 生命系统

- 经典 HP 条
- 能量电池回复
- 受击反馈（短暂无敌帧）

## 8. 敌人系统

### 8.1 阵营

纯敌对失控机械体，无同阵营竞争 AI。

### 8.2 二型制

| 类型 | 初始状态 | 索敌逻辑 | 攻击方式 |
|------|----------|----------|----------|
| **巡逻型** | 醒着巡逻 | 视觉锥范围内看到玩家 → 直接追击 | 近身攻击 |
| **休眠型** | 休眠不动 | 纯靠噪音范围触发 → 觉醒进入战斗 | 近身攻击，血量更高 |

### 8.3 初始分布

- 地图初始时在高价值区预置少量敌人（守卫者）
- 休眠型守在高价值容器附近，巡逻型在高价值区巡逻
- 低价值区初始敌人很少或没有

### 8.4 时间刷怪系统

敌人随时间推移从地图外缘 / 特定刷怪点持续涌入：

| 时间段 | 刷怪强度 | 敌人类型 |
|----------|----------|----------|
| 0-60s | 低 | 少量巡逻型 |
| 60-180s | 中 | 巡逻型 + 少量休眠型 |
| 180s+ | 高 | 大量混合，强度持续递增 |

刷怪规则：
- 从地图边缘 / 预设刷怪点生成，按密度分布
- 生成时屏幕边缘出现脉冲 UI 提示 + 方向音效
- 信号弹发射后刷怪强度瞬时跳升（大量敌人向信号点聚集）
- 敌人不会在玩家视野内凭空生成

### 8.5 行为摘要

| 类型 | 巡逻 | 追击 | 攻击 |
|------|------|------|------|
| 巡逻型 | 随机路线 | 看到玩家直接追击 | 近身攻击 |
| 休眠型（未觉醒） | ❤ 休眠 | ❤ | ❤ |
| 休眠型（觉醒） | ❤ | ✅ | 近身攻击 |

## 9. 地图系统

### 9.1 规模

**固定地图（20-30 屏）**，以 1920×1080 视野范围为单位。

### 9.2 地图设计

手工制作固定地图，随机性由事件系统提供：

- 固定布局：障碍物、地形、区域划分均为手工设计
- 容器位置固定，内容随机（每局刷新）
- 初始敌人位置固定（高价值区守卫）
- 刷怪点位置固定（地图边缘 + 特定区域）

### 9.3 事件系统

每局随机触发事件，为固定地图提供重玩随机性：

| 事件类型 | 示例 | 触发方式 |
|----------|------|----------|
| **容器警报** | 破解某容器触发警报，周围敌人警戒值暴涨 | 破解时随机触发 |
| **紧急补给** | 地图某处出现临时弹药/电池补给点 | 定时随机 |
| **电磁脉冲** | 某区域短暂断电，敌人全部瞬间觉醒 | 定时随机 |
| **封锁区域开启** | 原本封锁的捕获通道打开，暴露高价值容器 | 时间触发 |
| **敌人潮汐** | 某方向突然涌入大量敌人 | 时间触发 |

### 9.4 地图边界

方形地图。使用自然屏障围合：

| 边界类型 | 世界观解释 |
|----------|-----------|
| 电磁障壁 | 静默灾难残留电磁场，翡翠绿半透明光墙 |
| 裂谷/断层 | 数据腐败撕裂的地表裂缝 |

### 9.5 结构

- 起始投放点（边缘固定位置）→ 探索区 → 任意位置撤离
- 高/低价值区在地图中手工预设
- 刷怪点分布在地图边缘和特定位置

### 9.6 战争迷雾

- 圆形视野（视角跟随角色）
- 走过区域留痕（永久亮）
- 未探索区域完全不可见
- 侵蚀越高 → 视野半径越小

## 10. 撤离系统

### 10.1 信号弹撤离

- 地图上无固定撤离点
- 玩家在任何位置发射信号弹
- 母车《余晖号》飞抵信号位置（**60-90 秒**等待）
- 等待期间信号吸引附近机械体涌来，**可能生成新机械体**
- 母车到达后交互登车 = 撤离成功

### 10.2 撤离决策

- 撤离等待时间长 → 需要选一个相对安全的时机和位置
- 侵蚀越高 → 视野越小 → 坚守更难
- 信号弹发射 = 全图噪音 + 刷怪强度跳升 = 大量敌人向信号点聚集

---

## 11. 压力系统

### 11.1 残响负荷（Erosion）

侵蚀是全局生存压力，随存活时间持续增长，被攻击时跳升：

- **时间流逝**：每 X 秒侵蚀 +1%
- **受击**：被攻击侵蚀 +2~3%
- **净化剂**：使用降低 15~20%（局内医疗容器搜刮获得）

侵蚀影响：
- 视野范围↓、破解速度↓
- 敌人 HP/伤害↑、刷怪速度↑
- 刷怪池逐渐解锁更强敌人类型
- 100% 硬门槛：无法拾取任何物品（不致死）

### 11.2 时间刷怪压力

- 敌人数量和强度随时间持续递增
- 形成自然的时间压力：拖得越久越危险
- 代替了原风暴系统的压力源角色

### 11.3 压力协同

四重压力相互驱动：
1. 时间推移 → 侵蚀↑ + 刷怪强度↑ → 敌人更强更多
2. 受击 → 侵蚀跳升 → 加速进入高侵蚀状态
3. 战斗 → 噪音↑ → 警戒值积累 → 更多敌人觉醒
4. 信号弹 → 全图噪音 + 刷怪跳升 = 大量敌人向信号点聚集

| 物品 Weight/数值表 | 每种物品的负重、分值、补给量、回复量、净化量 |
| maxWeight/maxErosion 初始值 | 角色初始上限 |
| 侵蚀时间增长率 | 每 X 秒 +1% |
| 受击侵蚀跳升值 | 被攻击 +2~3% |
| 净化剂降低侵蚀值 | 15~20% |
| 侵蚀→破解读条时间映射 | 侵蚀值对应的破解时长 |
| 侵蚀→视野半径映射 | 侵蚀值对应的迷雾范围 |
| 侵蚀→敌人属性倍率 | 各侵蚀段的 HP/伤害倍率 |
| 侵蚀→刷怪间隔乘数 | 各侵蚀段的刷怪间隔乘数 |
| 各行动噪音值 + 传播范围 | 噪音源参数 |
| 各敌人警戒阈值 | 唤醒临界值 |
| 警戒值消退速率 | 噪音自然衰减速度 |
| 信号弹等待时间 | 60-90s |
| 刷怪时间曲线 | 各时间段刷怪数量/强度/间隔 |
| 事件触发概率 | 各事件的触发概率和时机 |
| 武器参数 | 伤害、射速、弹匣 |
| 弹药携带量 + 补给量 | 初始 60 发 + 补给盒数量 |
| 收集品分数表 | 各收集品撤出分数 |

## 16. 不做 (Out of Scope)

- 多角色选择
- 技能树/天赋系统

## 17. 升级方向 (Fallback / If Time Permits)

> 以下各项为设计过程中为控制 1 周 Scope 而选择的简化方案。如有空余时间，按优先级逐项升级。

### 优先级 P0：核心体验提升（预计各 0.5-1 天）

| 系统 | 当前方案 | 升级方向 | 收益 |
|------|----------|----------|------|
| **弹种切换** | 单弹种（标准弹） | → 轻型枪支持 标准弹 / 穿甲弹 / 静默弹 切换，切换需 Reload 动画（1-1.5s） | 「怎么打」的即时决策 |
| **敌人三型制** | 二型制（巡逻+休眠） | → 恢复中型（醒着驻守，宽锥 120° 视觉 + 警戒值觉醒），形成轻/中/重三型 | 战术差异感 |
| **噪音尖峰** | 无 | → 短时间连续开枪触发警戒值跳升 | 「要不要连射」的即时决策 |

### 优先级 P1：系统深度扩展（预计各 1-2 天）

| 系统 | 当前方案 | 升级方向 | 收益 |
|------|----------|----------|------|
| **局外装备选择** | 固定初始装备 | → Run 前选择初始携带（武器/弹药/消耗品），玩家决定进图带什么 | 策略准备阶段 |
| **防御装备** | 无 | → 护甲/护盾等防御装备，减伤或吸收伤害，增加装备决策维度 | 生存策略 |
| **投掷道具** | 无 | → 电磁诱饵、静默烟雾、脉冲手雷等投掷类道具 | 策略维度 +1 |
| **武器品质** | 单品质 | → 白/绿/蓝/紫四级品质（伤害/弹匣/噪音差异），局内武器残骸容器极小概率掉落 | 搜刮惊喜感 |
| **容器重封** | 破解后永久打开 | → 已破解未拾取容器 60-90s 后重新密封，再次破解需重新读条+侵蚀 | 决策紧迫感 |
| **搜刮惊喜** | 收集品+弹药 | → 残响遗物（单 Run 限时 buff：「30s 移速+15%」「下一发必暴击」） | 搜刮惊喜感 |

### 优先级 P2：Meta 与经济（预计 2-3 天）

| 系统 | 当前方案 | 升级方向 | 收益 |
|------|----------|----------|------|
| **Meta 成长** | 每 Run 独立，无跨 Run 进度 | → 收集品撤出后卖出换「残响晶片」；晶片解锁记忆碎片（条件性增益入池 + 微量永久数值） | Roguelite 长期动力 |
| **经济系统** | 收集品 = 分数 | → 收集品卖出换晶片，双消费路径（记忆解锁 / Run 前补给商店） | 资源管理深度 |
| **难度系统** | 单一难度 | → 三级难度（微弱回响 / 清晰信号 / 尖锐悲鸣），刷怪曲线/敌人密度分级 | 重玩价值 |
| **重型枪** | 无 | → Meta 解锁蓝图后可带入，弹药池为电磁弹/散射弹/高爆弹 | 武器多样性 |
| **局外界面** | 简单开始界面 | → 母车主界面 + 记忆碎片展厅 + 交易终端 + 补给商店 + 难度选择 | 沉浸感 |

### 优先级 P3：大赛后扩展

| 系统 | 当前方案 | 升级方向 |
|------|----------|----------|
| 角色 | 单一主角 | → 多猎人（不同初始技能、maxWeight/maxErosion 侧重） |
| 武器 | 轻型枪 | → 模组拼接式（枪身+枪管+弹药类型自由组合） |
| 敌人 | 二型制 | → 功能分化制（哨兵/突击者/炮台/干扰者） |
| 地图 | 固定单张 | → 多生物群落地图池（数据中心废墟 / 搁浅货轮 / 废弃高铁站） |
| 生命 | 经典 HP 条 | → 侵蚀增伤联动（侵蚀越高受到伤害越高） |
| 美术 | AI 像素风 | → HD-2D 后处理（Bloom + 色差 + 景深模糊） |
| 内容 | Demo 规模 | → BOSS 战、剧情分支、多结局 |
