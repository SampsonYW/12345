# 《余晖号》Afterglow Express

### 2D Roguelite × 搜打撤 (Search-Fight-Extract)

---

## 🎮 一句话介绍

> **在电磁废墟中搜刮旧文明遗物，与失控机械体周旋，在贪婪与生存之间做出抉择——然后发射信号弹，祈祷你能活着离开。**

---

## 📖 世界观

**2040 年代**，一场被称为「静默」的电磁灾难摧毁了全球通信。文明退回孤岛化，旧时代的科技残骸散落荒野，AI 和机械体脱离控制，形成了危险的「领地」。

你是**残响猎人**——驾驶拼装磁轨车《余晖号》，深入静默区搜寻旧世界遗物。这些残响碎片承载着文明的记忆，也是重建未来的钥匙。

**视觉调性**：翡翠绿 × 黄昏橙 · 近未来轻末世 · 硬科幻外壳 + 温情内核

---

## 🔁 核心循环：搜 → 打 → 撤

一次 Run 的完整体验（约 5 分钟）：

```
投放至静默区 → 探索 & 破解容器 → 遭遇机械体 → 权衡去留 → 发射信号弹 → 坚守等待 → 撤离/死亡
```

| 阶段 | 你在做什么 | 压力来自哪里 |
|------|-----------|-------------|
| 🔍 **搜** | 破解密封容器、搜刮遗物、管理负重和侵蚀 | 破解需要时间，侵蚀让视野缩窄 |
| ⚔️ **打** | 与失控机械体交战 | 枪声唤醒更多敌人，弹药越打越少 |
| 🚀 **撤** | 发射信号弹、坚守 60-90 秒等母车到达 | 信号吸引全图敌人涌来 |

**核心张力：你永远在"再搜一个"和"现在就撤"之间挣扎。**

---

## ✨ 核心创新

### 1. 双 Tag 物品系统

每件物品有两个代价维度：

| Tag | 含义 | 效果 |
|-----|------|------|
| **Weight（负重）** | 物理重量 | 拿满了就带不动更多 |
| **Erosion（侵蚀）** | 精神污染 | 随时间自然增长 + 受击跳升。侵蚀越高 → 视野越窄 + 破解越慢 + 敌人越强 + 刷怪越猛 |

**设计目的**：不是"拿不拿"，而是"值不值得承受代价"。

### 2. 破解 Loot + 净化剂

物品不是捡的——它们锁在密封容器里。

- 长按交互 **2-5 秒**读条破解（侵蚀越高越慢）
- 破解期间**不能移动、不能攻击**——把后背暴露给危险
- 被攻击则**读条中断**
- 搜刮**净化剂**可暂时降低侵蚀——但医疗容器稀少，且破解它也需要时间

**设计目的**：搜刮有代价也有解药，每一次蹲下破解都是在和时间赌博。

### 3. 独立警戒值

每个敌人拥有**独立的噪音警戒条**：
- 玩家的行动产生噪音 → 按距离衰减传播
- 噪音**累加**到敌人的警戒条 → 超阈值觉醒
- 警戒值会**缓慢消退**

**设计目的**：不是"被发现/没被发现"的二元判定，而是渐进式的紧张感积累。

### 4. 信号弹撤离

没有固定撤离点——你在任何位置发射信号弹，母车飞来接你。
但信号弹 = **全图噪音** + **刷怪强度暴增**，等待的 60-90 秒是最危险的时刻。

**设计目的**：撤离不是"跑到出口"，而是"选择在哪里坚守最后一战"。

### 5. 时间刷怪 + 事件系统

- 敌人随时间从地图边缘持续涌入，**强度递增**
- 固定地图 + **随机事件**（容器警报、紧急补给、电磁脉冲、封锁区开启）

**设计目的**：拖得越久越危险，同时每局体验不同。

## 🎯 压力协同

四重压力相互咬合，构成核心体验张力：

```
  ┌──────────────────────────────────────────┐
  │                                          │
  │   时间 ──→ 侵蚀↑ ──→ 视野↓ 破解↓ 敌人↑  │
  │        │                 刷怪↑           │
  │   受击 ──→ 侵蚀跳升                      │
  │        │                                 │
  │   战斗 ──→ 噪音↑ ──→ 警戒值↑ ──→ 更多敌人│
  │        │                                 │
  │   信号弹 ──→ 全图噪音 + 刷怪跳升         │
  │        │                                 │
  │   净化剂 ──→ 侵蚀↓（稀缺，需搜刮）        │
  │                 ↓                        │
  │            最终抉择：活着离开              │
  └──────────────────────────────────────────┘
```

**你做的每一件事都在让局势恶化。唯一的反制手段是搜索和撤退。**

## 🖼️ 视觉风格

| 要素 | 说明 |
|------|------|
| 视角 | 3D 正交斜俯视（参考 Don't Starve 的斜俯视读法） |
| 美术 | AI 辅助像素风（ComfyUI 本地生成） |
| 主色调 | 翡翠绿（电磁/科技）+ 黄昏橙（温暖/人性） |
| 战争迷雾 | 圆形视野锥，走过留痕，侵蚀影响视野大小 |
| 氛围参考 | 《星之终途》(Stella of the End) |

---

## 🎵 音效设计

| 阶段 | 音乐氛围 |
|------|----------|
| 探索 | 低频环境音，偶尔远处机械体的金属碰撞声 |
| 战斗 | 紧张电子鼓点，随敌人数量渐强 |
| 撤离等待 | 高压脉冲节拍，心跳感 |

关键音效反馈：
- 破解读条的电子嗡鸣
- 敌人觉醒时的机械启动声
- 信号弹升空的嘶鸣
- 母车《余晖号》到达时的磁轨轰鸣
